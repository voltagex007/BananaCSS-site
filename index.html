Banana CSS

A very decent looking CSS Framework Geared towards Doc Creators using only HTML and CSS

Features

Colors

Green and Black Colors are used to get clean in unique look for your document. Text selection does green text, try selecting this text. Also Buttons use Green and Black.

Ample of Animations and Transitions

Some good amount of Transitions and Animations are made for few components which deserve some. Like the button having colors ans border Transitions. Also they are visible and quick. It gives a salty Visual Feedback which is very good than just having a simple default button.

Font

The font 'Raleway' is used to get a matching Font as well as readable font.

Technology

Various modern Technologies, like SCSS and VSCode (with lots of Extensions) are used to develop, resulting in very rapid improvements, that will make bug fixes, new features get implemented really quick.

Get It

All you have to do is just download the small files and simply paste it inside your website's working directory (or anywehre else as per your preferences) and just use the following code inside the head tag in the html file

<link rel="stylesheet" src="Banana/banana.min.css">

The current version is "beta1", BananaCSS-beta2 is Work In Progress. beta1 is basically very basic and just works. It will have more features expected in the next few beta releases. This website uses the current beta version that can be viewed as an Example.

Get Started

Kickstarter

As a Kickstarter, I will be explaining how to get started. After when you have linked the css file, you can use its unique styles by referring those classes and use tags to fine finish your website.

A simple DOC Site

We will start from creating a Documentary site using BananaCSS.

Lets create a navbar to be able to give the ability of Navigation for your Client/User.

<nav class="navbar">
    <div class="nav-item">
        <a href="#" class="nav-brand">LOGO</a>
    </div>
    <div class="nav-item">
        <a href="#" class="nav-link">Home</a>
    </div>
    <div class="nav-item">
        <a href="#" class="nav-link">Index</a>
    </div>
    <div class="nav-item">
        <a href="#" class="nav-link">Buy PDF</a>
    </div>
</nav>

This will create a Navbar wth 3 buttons and a Logo

The navbar class defines it as a navbar with flex as display. The nav-item defines it as a navbar's item, that can contain something that is placed inside the navbar with some css styling. nav-link makes the anchor tag a navbar link that has another some specific css styles and transitions defining it as a clickable text.

Let's now create a Title showing the name of the project (for what this Doc is being created)

To do that simply create a h1 tag and type the name. Thats it

To create a button, just create a anchor tag and give it a class of "btn" and it will become a button, "btn-outline" class will give it a different look (give either "btn" or "btn-outline" to make it have different looks.

There we have 2 different types of Buttons, try hovering and clicking over them.


Solid Outline

It's recommended to add br tags before and after anchor tag with btn classes.
